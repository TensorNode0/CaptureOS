INTERPRETATION
============================================================
- Cloudflare Error 1034 = "Edge IP Restricted" — indicates the hostname
  is on Cloudflare's network but no valid origin/route is configured
  (or the route is misconfigured).
- captureagent.us is CLEARLY not linked to the Emergent deployment
  (govcon-workspace.emergent.host), which is the correct conclusion.
- HOWEVER, the failure mode is NOT a TLS handshake failure. Cloudflare
  has already provisioned a valid SSL cert (issued 2026-07-04, TODAY),
  which suggests the user's Entri linking flow was partially initiated
  but the origin route is not yet configured or has not yet propagated.

Fix path for the user:
  1. Complete the Link Domain → Entri flow in the Emergent UI.
  2. If already initiated: wait 5–15 min for propagation.
  3. Verify Cloudflare CNAME/origin points to govcon-workspace.emergent.host.
